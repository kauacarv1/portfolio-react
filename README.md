## Como rodar o projeto
Clone o repositório:
```bash
git clone https://github.com/seu-usuario/portfolio-react.git
```
Entre na pasta:
```bash
cd portfolio-react
```
Instale as dependências:
```bash
npm install
```
Execute o servidor local:
```bash
npm run dev
```
